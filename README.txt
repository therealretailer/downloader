# downloader
Py3 Image from CSV Downloader

Greetings to everyone, in case you need to bulk download images from your CSV file you can use this pretty little script.

HOW TO LAUNCH?:
To use this script you need installed python3 and PIP, you can download it from official site.
Also you need to install couple libraries:

pandas
requests
To instal it open terminal and run script:

pip install pandas
pip install requests
HOW TO PREPARE CSV:
All that you need Is set two columns:
'link' - the image URL's
'name' - name that files will be named after downloading.

There are sample of CSV, with formula that pluses extension to your filename.
